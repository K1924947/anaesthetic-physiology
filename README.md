# Anaesthetic Physiology — Interactive Teaching Tools

A collection of interactive, browser-based physiology visualisations built for doctors, anaesthetic trainees and medical students. No installation required — open directly in any browser.

---

## Tools

### 📈 Pressure–Volume Loop
An interactive cardiac PV loop with smooth animated transitions between physiological states.

**Features:**
- Animated morphing between modifier states (Normal, ↑/↓ Preload, ↑/↓ Afterload, ↑/↓ Contractility, Heart Failure)
- Clickable phase segments (Filling, Isovolumetric Contraction, Ejection, Isovolumetric Relaxation)
- Live stat readout: EDV, ESV, SV, EF, Peak P, ESP
- EDPVR and ESPVR reference lines
- Ghost normal loop for comparison
- Key Relationships, Valve Events, and Clinical Correlation panels

---

### 🩺 Arterial Pressure Waveform
An interactive arterial line waveform with animated transitions between haemodynamic states.

**Features:**
- Animated waveform morphing between states (Normal, ↑/↓ SVR, ↓ CO, Hypovolaemia, Stiff Aorta)
- Clickable waveform features (Upstroke, Peak SBP, Downslope, Dicrotic Notch, Diastolic Runoff, Trough DBP)
- Live stat readout: SBP, DBP, Pulse Pressure, MAP
- Link to PV Loop, Key Relationships, and Clinical Relevance panels
