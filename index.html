About
Built alongside preparation for anaesthetic training — Each tool reflects a topic worth understanding from first principles, not just memorising for an exam. These started as personal study tools — built because drawing something out forces you to actually understand it. If they're useful to others, all the better.
For whom
For anyone wanting to understand physiology properly — whether consolidating knowledge for clinical practice or preparing for postgraduate exams including the FRCA and USMLE Step 1. All tools run entirely in the browser — no account, no download.
